# Kinetic Pulse Smartwatch Website

This is a team project website for Kinetic Pulse, a smartwatch powered by motion-based charging and equipped with real-time language translation.

## Project Overview
Kinetic Pulse is designed to offer a seamless smartwatch experience that charges through user motion and supports real-time translation across multiple languages.

- HTML  
- CSS  
- JavaScript
